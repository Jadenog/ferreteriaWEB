Proyecto personal sobre una ferreteria ficticia.

en el proyecto se implementaron como:

- un carrousel, para ver las noticias
- se importo una ubicacion desde google maps
- Un login que dependiende del rol(admin/cliente), nos saldra una interfaz diferente

desde el rol admin podemos: 
- agregar productos
- editar producto
- eliminar producto
- comprar producto
- agregar noticias
- editar noticias
- eliminar noticias

desde cliente podemos:
 - comprar producto

 Ambos roles pueden buscar producto.
 El login esta protegido, de tal manera que no podemos accedar a las interface a menos de que iniciemos sesion,
 de lo contrario no sera posible, y no podemos iniciar sesion como cliente, y por url, ingresar a admin, o viceversa
